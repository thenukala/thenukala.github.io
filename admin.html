Coming Up — Home Page Panel

The "Coming Up" panel on the Home page auto-shows events, birthdays and anniversaries in the next 14 days. To populate it:

  • 🎂 Birthdays — set the Birthday field in Admin → Contacts for each member
  • 💍 Anniversaries — set the Anniversary field in Admin → Contacts
  • 📅 Events — add events in Admin → Events with a date

No manual editing needed — it updates automatically from your data.

Edit the eyebrow text, title and description shown at the top of each page.
