📝 How to Add More Family Members

Open tree.html in a text editor (Notepad on Windows).
To add a person: copy any <div class="person">...</div> block and paste it inside a <div class="family-unit">.
To add a spouse: wrap two person cards in a <div class="couple"> with <div class="heart-link">💛</div> between them.
To add a generation: copy an entire generation block and update the generation number, badge text, and person details.
To mark someone deceased: add the class deceased to their person div.
To mark someone as female: add the class female to their person div.
The tree supports unlimited generations — just keep adding! 🌳
